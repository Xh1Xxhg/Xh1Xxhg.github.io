# 发布到 Github Pages

npm install hexo-deployer-git --save

修改根目录下的 _config.yml ，配置 GitHub 相关信息

deploy:
  type: git
  repo: https://github.com/Xh1Xxhg/Xh1Xxhg.github.io.git
  branch: main
  token: ghp_xxxxxxxxxxxxxxxxxxx(自己生成)

token 生成

hexo g -d 部署

直接访问即可

https://<yourname>.github.io/
