# 网安学习建议

学习方法：

  • 实操最重要
  • 需要什么的时候就去搞什么，不要按照某些所谓的顺序学下去，避免热情消磨殆尽
  • 拒绝一味地使用工具

引用圈内大佬说的：

“知识面，决定看到的攻击面多广”

“知识链，决定发动的杀伤链多深”

# 前置必备

  1. CTF 方向：Web、MISC、Crypto、Reverse、PWN、AI Sec (偏科研)、Blockchain 等
  2. 编程语言：PHP、Python， Java，C/C++，汇编，Golang，合约语言…
  3. 建议用心学的本科课程：C 语言、微机原理、面向对象程序设计 (Java)、计算机网络、操作系统、信息安全数学基础、现代密码学、区块链应用、网络攻防…

# 推荐方向

①Web 漏洞 —— 应用灵活：Web + MISC + Crypto + Blockchain + ?Re…

②二进制底层漏洞 —— 积累的过程很枯燥：PWN + Re + Crypto

# CTF

多打比赛积累经验，基本上每段时间都有一堆比赛

  • NSSCTF
  • 攻防世界
  • CTF 赛事通知群 734535934
  • Hello CTF 非常好的学习网站，目前还在更新...

# Web

知识点非常杂，遇到不懂的浏览器直接搜效率最高

基础

  1. 科学上网
  2. Kali / Linux，搭建虚拟机
  3. Python 写脚本
  4. 前后端 (直接做 b 站项目去学上手最快不要看书)
  5. 网络协议，数据报文分析

进阶

  1. 多打靶场：CTF /top10 漏洞 DVWA / 红队 Hack The Box (进阶)
  2. 多看博客，公众号，b 站，...，跟踪复现漏洞和 CTF 赛题
  3. CNVD 漏洞证书，公益 / 教育 SRC，企业 SRC (￥)，国外 SRC ($)
  4. 参加国家 / 地方护网
  5. 红队：Java 安全，内网渗透，免杀，代码审计，云安全，...

推荐

【小迪安全】Web 安全 | 全栈渗透测试工程师 @2023 最新完结版缺点：耗时太长了

《Web 漏洞解析与攻防实战》，不看也行

# Web 逆向

JS 逆向，安卓逆向，IOS 逆向

# Crypto

直接刷 CTF 题

加解密

  1. 古典密码
  2. 现代密码

可证明安全

偏科研，《Introduction to Modern Cryptography》Jonathan Katz

# MISC

直接刷 CTF 题

压缩包，编码，图片，音频，电子文档，流量分析，取证

# Reverse & PWN

基础

  1. c 语言
  2. 汇编 极力推荐课程 《微机原理》
  3. PE/ELF 文件结构
  4. 查壳脱壳
  5. ...

推荐阅读

<u> 必学 </u>

  1. 《汇编语言》王爽
  2. 《加密与解密》
  3. 《CTF 竞赛权威指南 (PWN 篇)》

<u> 进阶 </u>

  1. 《逆向工程核心原理》
  2. 《C++ 反汇编与逆向》
  3. 《intel 汇编语言程序设计》
  4. 《Windows 程序设计》、《Windows 驱动开发技术详解》
  5. 《IDA Pro 权威指南》

工具

  1. IDA
  2. pwntools
  3. OllyDBG
  4. x64dbg
  5. gdb

# 区块链

开发

前端开发 (Vue 或 React)，智能合约开发

联盟链、DApp 开发

安全

智能合约的安全审计及漏洞利用

直接去看历史漏洞文档，学习如何去审计合约

需要了解

  1. 合约语言：Solidity
  2. MetaMask 钱包
  3. Remix 在线 IDE（这里同时推荐 vscode）
  4. IPFS
  5. 结合密码学 (哈希算法，椭圆曲线算法...)
