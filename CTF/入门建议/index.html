# 网安学习建议

学习方法：需要什么的时候就去搞什么，不要按照某些所谓的顺序学下去，避免热情消磨殆尽

# 前置必备

  1. CTF 方向分类：Web、Misc、Crypto、Reverse、PWN、AI Sec (偏科研)、Blockchain 等

  2. 前后端学习 (直接做小项目去学上手最快，感悟最深)

  3. 编程语言：PHP (过时但入门必备)、Python， Java， Golang，C/C++，Solidity…

  4. 建议好好学的本科课程：汇编语言、面向对象程序设计 (Java)、计算机网络、操作系统、信息安全数学基础、现代密码学、区块链应用、网络攻防…

# 推荐方向

①Web 漏洞 —— 应用灵活：Web+Misc+Crypto+Blockchain…

②二进制底层漏洞 —— 积累的过程很枯燥：PWN+Re+Crypto

# PWN & 逆向

web 方向有 js 逆向，安卓逆向，ios 逆向...

类似的，PWN 方向也需要有对底层调用的逆向能力

基础

  1. c 语言
  2. 汇编语言（能看懂就行，微机原理好好学）
  3. PE/ELF 文件结构
  4. 查壳脱壳

推荐阅读

  1. 《逆向工程核心原理》
  2. 《加密与解密》
  3. 《C++ 反汇编与逆向》
  4. 《CTF 竞赛权威指南（PWN 篇）》
  5. 《intel 汇编语言程序设计》
  6. 《Windows 程序设计》、《Windows 驱动开发技术详解》
  7. 《IDA Pro 权威指南》

工具

  1. OllyDbg
  2. x64dbg (开源调试器)
  3. IDA
  4. gdb (Linux 下调试使用)

# 区块链

开发

前端开发 (Vue 或 React)，智能合约开发

联盟链、DApp 开发

安全

智能合约的安全审计及漏洞利用

直接去看历史漏洞文档，学习如何去审计合约

需要了解

  1. 合约语言：Solidity
  2. MetaMask 钱包
  3. Remix 在线 IDE（这里同时推荐 vscode）
  4. IPFS
  5. 结合密码学 (哈希算法，椭圆曲线算法...)
