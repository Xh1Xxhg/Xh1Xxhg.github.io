安装开启 ssh 服务

apt-get update
apt-get install ssh
service ssh start
service postgresql start
vi /etc/ssh/sshd_config
将PermitRootLogin改为yes 使它能够使用密码进行登录
保存退出，重启
/etc/init.d/ssh restart
